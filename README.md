# rotate-array-to

Rotate an array to a given index

## Installation

    npm install rotate-array-to

## Usage

```typescript
import rotateArrayTo from "rotate-array-to";

const array = [ "apples", "oranges", "bananas", "peaches" ];

rotateArrayTo(array, 2); // [ "bananas", "peaches", "apples", "oranges" ]
```
